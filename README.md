# QALab — Releases

Public download page for the **QALab** desktop app (Windows).

This repository contains **installers only** — no source code. The main project is private.

## Download

Open **[Releases](https://github.com/alkalo/QALab-releases/releases/latest)** and download:

| File | Use |
|------|-----|
| `QALab-*-Setup.exe` | NSIS installer (recommended) |
| `QALab-*-Portable.exe` | Portable, no install |

## Updates

The desktop app checks this repository for automatic updates.

## Requirements

Windows 10/11 (64-bit), 8 GB RAM recommended. First run installs Ollama and downloads AI models (~4–8 GB).
